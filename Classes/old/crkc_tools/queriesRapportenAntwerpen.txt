Een aantal queries gebruikt om rapporten te maken voor Provincie Antwerpen
Aantal objecten per collectie: SELECT cc.idno, count(co.idno) FROM `ca_objects_x_collections` coxc, ca_objects co, ca_collections cc WHERE coxc.object_id = co.object_id and coxc.collection_id = cc.collection_id group by cc.idno
Aantal afbeeldingen per collectie:  SELECT ca.idno, count( ca.collection_id ) FROM `ca_objects_x_collections` coxc, ca_collections ca WHERE coxc.collection_id = ca.collection_id AND coxc.object_id IN (SELECT ca.row_id FROM ca_attributes ca, ca_attribute_values cav WHERE ca.attribute_id = cav.attribute_id AND cav.element_id =202 AND ca.table_num =57) GROUP BY ca.idno
Aantal afbeeldingen per fiche: select co.idno, count(ca.row_id) from ca_attributes ca, ca_attribute_values cav, ca_objects co  where ca.attribute_id = cav.attribute_id and cav.element_id = 202 and ca.table_num = 57 and row_id = co.object_id group by co.idno
Aantal ingevulde elementen per fiche: select co.idno, count(cav.element_id) from ca_attributes ca, ca_attribute_values cav, ca_objects co  where ca.attribute_id = cav.attribute_id and ca.table_num = 57 and row_id = co.object_id and co.deleted = 0 group by co.idno
Jongste en oudste aanpassing per collectie: select ca.idno, from_unixtime(MAX(log_datetime)), from_unixtime(MIN(log_datetime)) from ca_objects_x_collections oxc, ca_change_log ccl, ca_collections ca where logged_row_id = oxc.object_id and oxc.collection_id = ca.collection_id and logged_table_num = 57 group by ca.idno
